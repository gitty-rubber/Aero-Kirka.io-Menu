3. Manual install:

Script icon should show Kirka.io favicon in your extension list.

## Usage

- Load https://kirka.io/
- Press **P** → menu appears
- Toggle Chams / No Recoil / Bhop checkboxes
- Tweak colors, opacity, size with sliders/pickers
- Close with the **Close** button or press **P** again
- Hold **Space** with Bhop enabled → auto hop
